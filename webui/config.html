Matrix Connection
Homeserver URL
Your Matrix homeserver URL (e.g., https://matrix.org). Use HTTPS to avoid authentication failures.
Bot User ID
Full Matrix user ID for the bot account (e.g., @my-bot:matrix.org)
Access Token
Bot's Matrix access token. Generate via: curl -X POST https://your-server/_matrix/client/v3/login -H 'Content-Type: application/json' -d '{"type":"m.login.password","user":"bot","password":"secret"}'
Device ID
Matrix device identifier for this bot session
Bot Settings
Display Name
Name shown in Matrix rooms. Also used as the trigger prefix in group chats.
Agent Identity
Identity string used in system prompts when the bot forwards messages to Agent Zero
Agent Zero API Key
Auto-detected API Key
This key is auto-generated by Agent Zero and used by the bot to forward messages. It is automatically applied when services start — no manual configuration needed.

If the key shows as empty, restart Agent Zero first.
